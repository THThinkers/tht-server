## 프로젝트에서 사용하는 .env 설정들

### DEFAULT CONFIG

- APP_PORT 서버가 돌아갈 포트
- DB_URL 데이터베이스 URL
- DB_USERNAME 데이터베이스 ID
- DB_PASSWORD 데이터베이스 비밀번호
- SESSION_KEY 세션 SECRET 키

### AUTHENTICATION

- GAUTH_CLIENT_ID 구글 로그인 API 키
- GAUTH_CLIENT_SECRET 구글 로그인 API SECRET 키
- GAUTH_REDIRECT_URL 구글 로그인 API 리다이렉트 주소 (구글 API 설정에 있음)
